Sosialisasi Aplikasi MyBest (E-learning)

Untuk itu mahasiswa baru diharapkan mengikuti sosialisasi penggunaan aplikasi MyBest secara langsung (live streaming) melalui kanal Youtube Universitas Bina Sarana Informatika.
